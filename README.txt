  CONTROLS:

	- Direction arrows of keyboard:
	     - Down: moves the ball upwards at the start of the game
	     - Left: activates the left flipper
	     - Right: activates the right flipper
	- ESC: exit the game

  MODO DEBUG:

	- F1: activates/deactivates the shapes silhouettes
	
  GAME DESCRIPTION:

	In this game you are able to play a version of Glamour Pinball developed by Suspending For Sure
	You have to try acomplish the highest score interacting, whith various things in the environment, as bumbers, signs and stars.
